# <a name="management-agent-run-error-codes"></a>Beheeragent foutcodes uitvoeren

De volgende tabellen bevatten mogelijk weergegeven in de gebruikersinterface Synchronization Service Manager in Microsoft Identity Manager (MIM) 2016 foutcodes en beschrijvingen voor elk van deze fouten.

## <a name="connection-errors"></a>Verbindingsfouten

| Fout                  | Beschrijving                                    |
|--------------------------|--------------------------------------------|
| kan geen verbinding =         | Verbinding met de gekoppelde map is mislukt voor een reden dan verificatie. Bijvoorbeeld, het netwerk is niet beschikbaar of de doelserver is offline.                                                                                                                                                                                                                                                                                                                      |
| verbroken verbinding        | De verbinding tussen de beheeragent en de gekoppelde adreslijst niet meer bestaat. De beheeragent probeert opnieuw verbinding maken met de gekoppelde adreslijst in veel gevallen.                                                                                                                                                                                                                                                                                                         |
| kan geen verificatie     | Verificatie is niet mogelijk met de opgegeven referenties.                                                                                                                                                                                                                                                                                                                                                                                                                          |
| kan geen machtiging         | Onvoldoende rechten voor toegang tot een container in de gekoppelde adreslijst. Deze fout worden alleen verwacht voor Lightweight Directory Access Protocol (LDAP) management agents die verschillende verbonden directory-containers doorzoeken.                                                                                                                                                                                                                                                              |
| kan niet zoeken            | Zoekactie voor een container of de tabel is mislukt met een onverwachte fout.                                                                                                                                                                                                                                                                                                                                                                                                                            |
| Waarschuwing-Nee-watermerk      | De beheeragent kan het watermerk niet lezen bij het uitvoeren van een volledige import. Deze fout worden alleen verwacht voor de beheeragent voor Sun één Directory Server 5.1 (voorheen iPlanet adreslijstserver) wanneer de eerste configuratie van beheeragent is voltooid en de gekoppelde adreslijst wijzigingenlogboek ingeschakeld heeft. Later, wanneer het wijzigingenlogboek gekoppelde adreslijst is uitgeschakeld, als de configuratie van de management-agent niet is bijgewerkt, deze waarschuwing treedt op wanneer een volledige import wordt uitgevoerd. |
| Er is geen start-partitie verwijderen | De LDAP-partitie oorspronkelijk is geconfigureerd voor deze agent management niet meer bestaat. Deze fout wordt geretourneerd wanneer de partitie is verwijderd of de partitie is verwijderd en opnieuw gemaakt met dezelfde naam. In het laatste geval ondanks dat de naam hetzelfde is, de fout partitie GUID wordt gewijzigd.                                                                                                                                                                           |
